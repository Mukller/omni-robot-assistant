# Датчики — Характеристики

## BNO055 IMU (9-DOF)

Инерциальный измерительный блок от Bosch, используется для измерения ориентации и угловых скоростей.

| Параметр | Значение |
|----------|----------|
| Модель | BNO055 |
| Оси | 9-DOF (акселерометр + гироскоп + магнетометр) |
| Режимы работы | IMU, COMPASS, M4G, NDOF, Fusion |
| Диапазон акселерометра | ±2/4/8/16g |
| Диапазон гироскопа | ±125–2000 °/с |
| Интерфейс | I²C (адрес 0x28/0x29), UART |
| Напряжение питания | 2.4–3.6В |
| Ток потребления | ~12.3 мА в режиме NDOF |
| Размер (плата Adafruit) | 20 × 27 мм |

### Подключение

```
BNO055 (Adafruit)    ESP32
-----------------    -----
VIN              --> 3.3V
GND              --> GND
SDA              --> GPIO21
SCL              --> GPIO22
```

### Режим NDOF (Absolute Orientation)

В режиме NDOF сенсор самостоятельно выполняет сенсорное слияние и выдаёт кватернион ориентации.
Калибровка выполняется автоматически при движении:

1. **Гироскоп** — откалиброван при неподвижном состоянии
2. **Акселерометр** — покрутить в 6 позициях
3. **Магнетометр** — движение восьмёркой

Статус калибровки: 0 (не откал.) → 3 (полностью откал.) для каждого сенсора.

```cpp
uint8_t sys, gyro, accel, mag;
bno.getCalibration(&sys, &gyro, &accel, &mag);
// sys == 3: полная калибровка системы
```

## INA219 Current Sensor

Измеряет ток и напряжение через shunt-резистор (I²C).

| Параметр | Значение |
|----------|----------|
| Модель | INA219 |
| Диапазон тока | 0 – 3.2А (shunt 100мОм) |
| Точность | ±0.5% |
| Диапазон напряжения шины | 0 – 26В |
| Интерфейс | I²C |
| Адрес | 0x40, 0x41, 0x44, 0x45 (конфигурируется A0/A1) |
| Напряжение питания | 3–5.5В |
| Разрешение тока | ~0.8 мА |

### Адресация

```
INA219-L (левый мотор):  A0=GND, A1=GND → адрес 0x40
INA219-R (правый мотор): A0=VCC, A1=GND → адрес 0x41
```

### Применение

- Мониторинг тока моторов (детекция перегрузки, заклинивания)
- Отображение в телеметрии ROS2 (`/battery_state` topic)
- Автоматическая остановка при превышении 2.5А (Safety класс, `OVERCURRENT_A`)

```cpp
CurrentSensor csL(0x40), csR(0x41);

float currentL = csL.getCurrent_mA() / 1000.0f;  // → Амперы
float voltageL = csL.getVoltage_V();

// Safety check
if (currentL > 2.5f || currentR > 2.5f) {
    motorL.stop(); motorR.stop();  // overcurrent protection (OVERCURRENT_A = 2.5A)
}
```

## ACS712 (альтернатива INA219)

Если INA219 недоступен, можно использовать аналоговый датчик тока ACS712.

| Параметр | ACS712-5A | ACS712-20A |
|----------|-----------|------------|
| Диапазон | ±5А | ±20А |
| Чувствительность | 185 мВ/А | 100 мВ/А |
| Интерфейс | Аналоговый (ADC) | Аналоговый (ADC) |
| Точность | ±1.5% | ±1.5% |

## Аккумулятор LiPo 3S

| Параметр | Значение |
|----------|----------|
| Химия | LiPo (Li-Polymer) |
| Конфигурация | 3S (3 ячейки последовательно) |
| Напряжение | 11.1В (номинальное) |
| Напряжение заряженного | 12.6В |
| Напряжение разряженного | 10.5В (порог отключения) |
| Ёмкость | 3000 мАч |
| Ток разряда | 25C = 75А пиковый |
| Разъём | XT60 |
| Зарядка | балансировочное зарядное устройство |

### Мониторинг заряда

```cpp
// Делитель напряжения: 100кОм + 33кОм → ratio = (100+33)/33 = 4.03
// ADC ESP32: 0–3.9В (ADC_11db) → GPIO36 (VP, input-only)
float voltage = analogRead(BATT_PIN) * 3.3f / 4095.0f * 4.03f;
int percent = (int)((voltage - 10.5f) / (12.6f - 10.5f) * 100.0f);
```

Низкий заряд (< 10.5В) → LED-индикация + тема `/battery_state` с `percentage < 0.2`.
