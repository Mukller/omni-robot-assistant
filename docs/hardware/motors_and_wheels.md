# Моторы и колёса — Характеристики

## JGA25-370 DC Gear Motor

Основной приводной мотор платформы Loki 200mm.

| Параметр | Значение |
|----------|----------|
| Модель | JGA25-370 |
| Напряжение | 12В DC |
| Передаточное число | 40:1 |
| Скорость без нагрузки | ~350 RPM @ 12V |
| Крутящий момент | ~3 кг·см |
| Ток холостого хода | ~70 мА |
| Ток при максимальной нагрузке | ~1.2 А |
| Энкодер | 48 CPR (квадратурный) |
| Тиков за оборот выходного вала | 48 × 40 = **1920 тиков/об** |
| Диаметр вала | 6 мм (с лыской) |
| Диаметр корпуса | 25 мм |

### Расчёт скорости

```
Длина окружности колеса = 2π × 37.5mm = 235.6 мм
Максимальная скорость ≈ 350 RPM × 235.6 мм ≈ 1.37 м/с
Рабочая скорость (50% PWM) ≈ 0.3–0.5 м/с
```

### Подключение к L298N

```
Мотор L         L298N          ESP32
-------         -----          -----
M+ (красный) -> OUT1           
M- (чёрный)  -> OUT2
             -> IN1  <-------- GPIO25
             -> IN2  <-------- GPIO26  
             -> EN A <-------- GPIO27 (PWM)

Мотор R         L298N          ESP32
-------         -----          -----
M+ (красный) -> OUT3
M- (чёрный)  -> OUT4
             -> IN3  <-------- GPIO14
             -> IN4  <-------- GPIO12
             -> EN B <-------- GPIO13 (PWM)
```

### Энкодер

```
Энкодер L          ESP32
---------          -----
VCC (3.3В)  -----> 3.3V
GND         -----> GND
A (жёлтый)  -----> GPIO32
B (белый)   -----> GPIO33

Энкодер R          ESP32
---------          -----
VCC (3.3В)  -----> 3.3V
GND         -----> GND
A (жёлтый)  -----> GPIO34
B (белый)   -----> GPIO35
```

> GPIO34/35 на ESP32 — только вход (input only), идеально для энкодеров.

## Drive Wheels (75mm rubber)

| Параметр | Значение |
|----------|----------|
| Диаметр | 75 мм (радиус 37.5 мм) |
| Диаметр отверстия вала | 6 мм |
| Материал | резина (антислип) |
| Диск | пластик |

Используется в конфигурации **дифференциального** привода (два мотора + кастер).

## Caster Wheel (шарик-кастер)

| Параметр | Значение |
|----------|----------|
| Диаметр шарика | 15 мм |
| Высота | ~20 мм |
| Нагрузка | до 1 кг |

Устанавливается в передней части шасси для баланса (3-я точка опоры).

## L298N Motor Driver

| Параметр | Значение |
|----------|----------|
| Напряжение логики | 5В |
| Напряжение питания моторов | 6–35В |
| Максимальный ток (один канал) | 2А |
| Пиковый ток | 3А |
| Тип управления | PWM через EN pin |
| Встроенный стабилизатор 5В | Есть (до 36В питания) |

### Ограничения

- Не рекомендуется для > 2А продолжительного тока → следить за нагревом
- Падение напряжения на драйвере ~2В → при 12В питании мотор получает ~10В
- INA219 измеряет ток после L298N для мониторинга
