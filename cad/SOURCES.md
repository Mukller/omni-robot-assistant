# CAD & 3D Print Sources

## Robot Frame
- **MakersPet Loki 200mm** — [GitHub](https://github.com/makerspet/makerspet_loki)
  Primary platform — differential drive with caster, 200mm diameter
  STL files: Bottom/Middle/Upper plates, Motor bracket ×2, Head, LDS02RR plate, RPLIDAR A1 plate

## LiDAR Mounts
- **Xiaomi LDS-01 / LDS02RR** — included in Loki STL set
- **RPLIDAR A1** — included in Loki STL set
- **YDLidar X4** — custom bracket (see `stl/ydlidar_x4_bracket.stl`)

## Reference Designs
- Thingiverse #515831 — universal LiDAR mount
- Thingiverse #2333324 — cable management clips for 200mm chassis
- GrabCAD — L298N mounting bracket templates
